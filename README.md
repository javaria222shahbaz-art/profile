#  Personal Profile Card

A modern and responsive **Personal Profile Card** built using **HTML5** and **CSS3** as part of the NeuroFive Solutions Frontend Web Development Internship - Week 1 Task.

# Project Overview

This project demonstrates the use of semantic HTML and modern CSS to create a clean, responsive, and visually appealing personal profile card. It focuses on building a solid webpage structure without using JavaScript or CSS frameworks.

# Features

- Semantic HTML5 structure (`header`, `section`, `footer`)
- Responsive design (optimized for 350px–500px screen widths)
- Modern glassmorphism UI
- Animated gradient background
- Profile photo section
- About Me section
- Career Goal section
- Skills badges
- Social media links with icons
- Reflection section
- Hover effects and smooth styling
- Built using only HTML & CSS

# Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome

# Learning Outcomes

During this project, I learned how to:

- Build webpages using semantic HTML.
- Apply the CSS Box Model effectively.
- Use margin, padding, border-radius, and box-shadow.
- Create responsive layouts.
- Improve webpage aesthetics using typography and modern UI design.
